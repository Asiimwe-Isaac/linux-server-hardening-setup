# linux-server-hardening-setup

A practical Linux security hardening project as part of my journey.

## Objective
This project demonstrates essential Linux server hardening techniques following security best practices. The skills practiced here (SSH hardening, firewall configuration, intrusion prevention, auditing) are directly applicable to securing cloud instances (AWS EC2, Azure VMs, GCP Compute Engine).

## Technologies & Tools Used
- **OS**: Ubuntu 24.04 LTS Server
- **Firewall**: UFW (Uncomplicated Firewall)
- **SSH**: OpenSSH with key-based authentication only
- **Intrusion Prevention**: Fail2Ban
- **Auditing & Scanning**: auditd, Lynis (optional: OpenSCAP + CIS Benchmark)
- **Automation**: Bash
